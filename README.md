## DNA Engineering Full-Stack Assignment
Build a CSV Parser.

## Prerequisites
- Java 17
- Node Js v20.10.0


## Description
CSV parser using Java/Spring Boot (+ unit tests using JUnit 5 and Mockito), and React.

### To access to the application 
Run Backend on localhost:8080 (I used MariaDB database to store data. Take a look at application.properties.)
Run Frontend on localhost:3000
For the CSV file use the file in data directory : data/employees.csv
